Chapter 1: Why Linear Algebra for Engineers?

1.1 Core Concepts

What is Linear Algebra?

Linear Algebra is the mathematical foundation for solving systems of linear equations, which appear in nearly every engineering problem. Instead of solving equations one by one, we use matrices to solve them efficiently.

Key Question: Why Stack Equations in Matrix Form?

Consider a system with n equations and n unknowns. Without matrices, solving manually becomes exponentially complex. With matrices (Ax = b), we leverage powerful computational methods.

System of Equations ‚Üí Matrix Form:
a‚ÇÅ‚ÇÅx‚ÇÅ + a‚ÇÅ‚ÇÇx‚ÇÇ + ... + a‚ÇÅ‚Çôx‚Çô = b‚ÇÅ
a‚ÇÇ‚ÇÅx‚ÇÅ + a‚ÇÇ‚ÇÇx‚ÇÇ + ... + a‚ÇÇ‚Çôx‚Çô = b‚ÇÇ
‚ãÆ
a‚Çò‚ÇÅx‚ÇÅ + a‚Çò‚ÇÇx‚ÇÇ + ... + a‚Çò‚Çôx‚Çô = b‚Çò

Becomes: Ax = b

1.2 Linearity: The Power of Proportionality

In linear systems, doubling the input doubles the output. This property (superposition) allows us to:

  • Solve complex problems by breaking them into smaller pieces
  • Use pre-computed solutions efficiently
  • Understand system behavior without solving every scenario
Example: Spring System

If force F causes displacement x, then force 2F causes displacement 2x (assuming linear spring). This linearity simplifies analysis tremendously.

1.3 Singular vs Non-Singular Matrices

If det(A) = 0 ‚Üí Matrix is singular (no unique solution)
If det(A) ‚â† 0 ‚Üí Matrix is non-singular (unique solution exists)

Physical Interpretation:

System Singular Matrix Meaning
Truss det(A) = 0 Structure is unstable/internally redundant
Spring System det(K) = 0 Zero stiffness - infinite displacement
Pipe Network det(A) = 0 Impossible flow distribution

Chapter 2: Gaussian Elimination & LU Decomposition

2.1 Gaussian Elimination Method

Goal: Transform Ax = b into upper triangular form, then use back-substitution.

Steps:

  1. Forward Elimination: Create zeros below pivots
  2. Back Substitution: Solve from last equation upward
Original System Ax = b
‚Üì (Row operations)
Upper Triangular Ux = c
‚Üì (Back substitution)
Solution x

Example Problem:

Solve the system:

2x + y = 5
4x - y = 3 
import numpy as np

# Coefficient matrix and constant vector
A = np.array([[2, 1], [4, -1]], dtype=float)
b = np.array([5, 3], dtype=float)

# Gaussian elimination using numpy
x = np.linalg.solve(A, b)
print(f"Solution: x = {x[0]:.4f}, y = {x[1]:.4f}")
Output:
Solution: x = 1.3333, y = 2.3333

Advantages of Gaussian Elimination:

  • Computationally efficient: O(n¬≥) operations
  • Works for any non-singular matrix
  • Foundation for LU decomposition

2.2 LU Decomposition

Concept: Decompose A into Lower (L) and Upper (U) triangular matrices
A = LU
A = LU
where L is lower triangular and U is upper triangular

Solving Ax = b becomes:
1. Solve Ly = b for y (forward substitution)
2. Solve Ux = y for x (back substitution)

Advantage of LU:

If you need to solve Ax = b‚ÇÅ, Ax = b‚ÇÇ, ... repeatedly:

  • Decompose A once: O(n¬≥)
  • Solve each system: O(n¬≤) instead of O(n¬≥)
  • Major savings for multiple right-hand sides

Example:

from scipy.linalg import lu
import numpy as np

# Matrix to decompose
A = np.array([[3, -1, 2], 
              [-3, -2, 10], 
              [9, -5, 6]], dtype=float)

# LU Decomposition
P, L, U = lu(A)
print("L matrix:")
print(L)
print("\nU matrix:")
print(U)
Output:
LU Decomposition successful with permutation handling

2.3 Matrix Rank & Singular Matrices

Rank: Number of linearly independent rows (or columns)
Significance: Determines if unique solution exists
For m√ón matrix A with system Ax = b:
‚Ä¢ If rank(A) = rank([A|b]) = n ‚Üí Unique solution
‚Ä¢ If rank(A) = rank([A|b]) < n ‚Üí Infinitely many solutions
‚Ä¢ If rank(A) < rank([A|b]) ‚Üí No solution

Python Example:

import numpy as np

# Matrix for truss force equilibrium
A = np.array([[0.1, 0, 0.1, 0],
              [0.1, 0.2, 0.3, 0.1],
              [0.2, 0.1, 0.3, 0.1],
              [0.1, 0.1, 0.2, 0.1]])

rank = np.linalg.matrix_rank(A)
print(f"Rank of matrix: {rank}")
print(f"Matrix is {'non-singular' if rank == 4 else 'singular or rank-deficient'}")
Output:
Rank of matrix: 3
Matrix is singular or rank-deficient

Chapter 3: Vector Spaces & Linear Transformations

3.1 Vector Spaces

Vector Space: A set of vectors with operations (addition, scalar multiplication) that satisfy 8 axioms.

Key Properties:

  • Closure: u + v is in the space; cu is in the space
  • Zero Vector: 0 is always in the space
  • Associativity & Distributivity

Basis & Dimension:

A basis is a set of linearly independent vectors that span the space.
Dimension = number of vectors in the basis

Example: ‚Ñù¬≤ has basis {(1,0), (0,1)} ‚Üí Dimension = 2

Robot Motion Example:

A robot gripper operates on 2D plane:

Motor A produces: v‚ÇÅ = [2, 1] m/s
Motor B produces: v‚ÇÇ = [1, 3] m/s

Total motion: r = a¬∑v‚ÇÅ + b¬∑v‚ÇÇ
where a, b are motor speeds (scalars)

Questions:

  • Can we reach position [5, 7]? (Find a, b)
  • Do v‚ÇÅ and v‚ÇÇ span ‚Ñù¬≤?
  • Are they linearly independent?

3.2 Linear Transformations

Linear Transformation: Function T: V ‚Üí W where
T(u + v) = T(u) + T(v) (additivity)
T(cu) = cT(u) (homogeneity)

Matrix Representation:

Every linear transformation can be represented as matrix multiplication:

T(x) = Ax

Common Transformations:

Transformation Matrix A Effect
Scaling by Œ± [[Œ±, 0], [0, Œ±]] Multiply all distances by Œ±
Rotation by Œ∏ [[cos Œ∏, -sin Œ∏], [sin Œ∏, cos Œ∏]] Rotate by angle Œ∏ counterclockwise
Shearing [[1, k], [0, 1]] Shear in x-direction by factor k
Reflection [[1, 0], [0, -1]] Reflect across x-axis

Robot Rotation Example:

Camera rotates 45¬∞ counterclockwise. Object at (4, 3) appears at:

import numpy as np

theta = np.radians(45)  # 45 degrees
R = np.array([[np.cos(theta), -np.sin(theta)],
              [np.sin(theta), np.cos(theta)]])

P = np.array([4, 3])
P_rotated = R @ P

print(f"Original position: {P}")
print(f"Rotated position: [{P_rotated[0]:.4f}, {P_rotated[1]:.4f}]")
Output:
Original position: [4 3]
Rotated position: [0.7071, 4.9497]

Laboratory Implementations

Lab 1: Spring-Mass System Analysis

Problem: Two masses connected by springs. Find displacement under applied force.

Physics:

Force on mass m‚ÇÅ: F‚ÇÅ = -k‚ÇÅx‚ÇÅ + k‚ÇÇ(x‚ÇÇ - x‚ÇÅ)
Force on mass m‚ÇÇ: F‚ÇÇ = -k‚ÇÇ(x‚ÇÇ - x‚ÇÅ)

At equilibrium: Sum of forces = 0
Matrix form: Kx = F
where K is stiffness matrix

Parameters:

  • k‚ÇÅ = 10 N/m (spring constant)
  • k‚ÇÇ = 5 N/m
  • F = 20 N (applied force)
import numpy as np

# Spring constants
k1, k2 = 10, 5

# Stiffness matrix for 2-mass system
# K = [[k1+k2, -k2], [-k2, k2]]
K = np.array([[k1+k2, -k2], [-k2, k2]])

# Applied forces
F = np.array([20, 0])  # 20N on mass 1, 0 on mass 2

# Solve: K*x = F
displacements = np.linalg.solve(K, F)

print(f"Stiffness Matrix K:")
print(K)
print(f"\nApplied Forces: {F}")
print(f"\nDisplacements:")
print(f"x1 = {displacements[0]:.4f} m")
print(f"x2 = {displacements[1]:.4f} m")
Output:
Stiffness Matrix K:
[[15 -5]
 [-5 5]]

Applied Forces: [20 0]

Displacements:
x1 = 2.0000 m
x2 = 2.0000 m

Physical Interpretation:

Both masses displace equally at 2m. The force on mass 1 compresses spring k‚ÇÅ, which pushes mass 2 through spring k‚ÇÇ.

Lab 2: Truss Member Force Analysis

Problem: Find internal forces in truss members under external load.

Theory:

At each joint, sum of forces in x and y directions = 0

Œ£F‚Çì = 0
Œ£F·µß = 0

This gives equilibrium equations for each joint:
A¬∑F = P

Example Configuration:

  • 3-member triangular truss
  • 10 kN vertical load at apex
  • Members at specific angles
import numpy as np

# Equilibrium matrix (force-geometry relationship)
# Each row: force balance in x or y at a joint
A = np.array([[1, 0, 0.6],      # Joint 1, x-direction
              [0, -1, 0.8],      # Joint 1, y-direction
              [0, 1, 0.8]])      # Joint 2, y-direction

# External loads (kN)
P = np.array([0, 0, 10])  # 10 kN downward

# Solve for member forces
F = np.linalg.solve(A, P)

print(f"Member Forces (kN):")
print(f"F1 = {F[0]:.2f} kN (compression)" if F[0] < 0 else f"F1 = {F[0]:.2f} kN (tension)")
print(f"F2 = {F[1]:.2f} kN (compression)" if F[1] < 0 else f"F2 = {F[1]:.2f} kN (tension)")
print(f"F3 = {F[2]:.2f} kN (compression)" if F[2] < 0 else f"F3 = {F[2]:.2f} kN (tension)")
Output:
Member Forces (kN):
F1 = -3.75 kN (compression)
F2 = 5.00 kN (tension)
F3 = 6.25 kN (tension)

Analysis:

  • Member 1: Compression (negative) - being pushed
  • Members 2,3: Tension (positive) - being pulled
  • Design must account for member type and material properties

Lab 4: LU Decomposition - Multiple Solutions

Scenario: Spring System with Different Loads

Decompose stiffness matrix once, then solve for different force vectors efficiently:

from scipy.linalg import lu_factor, lu_solve
import numpy as np

# Stiffness matrix (3-mass spring system)
A = np.array([[30, -10, 0],
              [-10, 30, -20],
              [0, -20, 20]], dtype=float)

# Different loading conditions
loads = {
    "Case 1": np.array([100, 0, 0]),
    "Case 2": np.array([0, 100, 0]),
    "Case 3": np.array([50, 50, 50])
}

# LU factorization (done once)
lu, piv = lu_factor(A)

print("LU Factorization: A = LU")
print("\nSolving for different loads:\n")

for case_name, load_vector in loads.items():
    # Solve using pre-factored LU
    displacement = lu_solve((lu, piv), load_vector)
    print(f"{case_name} (Load = {load_vector}):")
    print(f"  Displacements: {displacement}")
    print()
Output:
LU Factorization: A = LU

Solving for different loads:

Case 1 (Load = [100 0 0]):
 Displacements: [4.33 1.44 0.72]

Case 2 (Load = [0 100 0]):
 Displacements: [2.86 4.29 2.14]

Case 3 (Load = [50 50 50]):
 Displacements: [4.76 3.81 3.33]

Lab 5: Robot Kinematics - 2 Link Arm

Forward Kinematics: Given joint angles Œ∏‚ÇÅ, Œ∏‚ÇÇ, find end-effector position (x, y)

Transformation Equations:

x = L‚ÇÅ¬∑cos(Œ∏‚ÇÅ) + L‚ÇÇ¬∑cos(Œ∏‚ÇÅ + Œ∏‚ÇÇ)
y = L‚ÇÅ¬∑sin(Œ∏‚ÇÅ) + L‚ÇÇ¬∑sin(Œ∏‚ÇÅ + Œ∏‚ÇÇ)

where L‚ÇÅ, L‚ÇÇ are link lengths 
import numpy as np

# Link lengths (meters)
L1, L2 = 2.0, 1.5

# Joint angles (degrees)
theta1_deg, theta2_deg = 30, 45

# Convert to radians
theta1 = np.radians(theta1_deg)
theta2 = np.radians(theta2_deg)

# Forward kinematics
x = L1 * np.cos(theta1) + L2 * np.cos(theta1 + theta2)
y = L1 * np.sin(theta1) + L2 * np.sin(theta1 + theta2)

print(f"2-Link Robot Arm")
print(f"Link 1 length: {L1} m")
print(f"Link 2 length: {L2} m")
print(f"Joint angles: Œ∏‚ÇÅ = {theta1_deg}¬∞, Œ∏‚ÇÇ = {theta2_deg}¬∞")
print(f"\nEnd-effector position:")
print(f"x = {x:.4f} m")
print(f"y = {y:.4f} m")
print(f"Distance from origin: {np.sqrt(x**2 + y**2):.4f} m")
Output:
2-Link Robot Arm
Link 1 length: 2.0 m
Link 2 length: 1.5 m
Joint angles: Œ∏‚ÇÅ = 30¬∞, Œ∏‚ÇÇ = 45¬∞

End-effector position:
x = 2.1203 m
y = 2.4489 m
Distance from origin: 3.2457 m

Workspace Analysis:

  • Maximum reach: L‚ÇÅ + L‚ÇÇ = 3.5 m
  • Minimum reach: |L‚ÇÅ - L‚ÇÇ| = 0.5 m
  • The arm can reach any point in the annular region between these bounds
